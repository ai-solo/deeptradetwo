## 测试工具特性

### 实时数据展示
- 连接到 feeder_client WebSocket (端口 9020)
- 使用 **JSON 格式**订阅数据（高实时性）
- 实时打印接收到的市场数据
- 每 10 秒显示统计信息

### 订阅内容
- `4.23.*` - 上交所债券快照
- `6.51.*` - 深交所债券快照
- `6.54.*` - 深交所债券其他

```
[时间] 市场 | 证券代码 | 证券名称 | 价格 | 涨停价 | 跌停价

```
```
[18:45:23.123] 上交所债券 | 010609 | 06国债09 | 价格: 100.000 | 涨停: 110.000 | 跌停: 90.000
```

## 故障排查

### 1. 连接失败
```
连接失败: dial tcp 47.101.149.89:9020: connect: connection refused
```
**原因**: feeder_client 未运行或端口未开放
**解决**:
- 检查服务器上 feeder_client 是否运行：`ps aux | grep feeder_client`
- 检查端口是否监听：`netstat -tlnp | grep 9020`
- 检查防火墙/安全组是否开放 9020 端口

### 2. 订阅成功但无数据
```
✓ 订阅请求已发送
...
(长时间无数据推送)
```
**原因**:
- **非交易时间** - 债券市场可能已收盘
- **周末/节假日** - 市场休市
- **数据权限** - Token 可能只有部分债券权限

**解决**:
- 在交易日 9:30-15:00 测试
- 使用通配符测试所有数据：`{"format":"csv","subscribe":["*.*.*"]}`

### 3. 数据格式错误
```
[解析错误] {"sid":4,"mid":23,"lt":"...",...}
```
**原因**: JSON 数据格式与预期不符
**解决**: 检查 feeder_client 日志，确认数据格式
